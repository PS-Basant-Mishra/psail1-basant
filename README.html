RAG Assistant POC

A minimalist, retrieval-augmented intelligence system capable of grounded document reasoning.

01 Quick Start

pip install -r requirements.txt
pip install faiss-cpu ollama
python app.py

Default access: localhost:5001

02 Config

  • Set API_KEY in app.py
  • Model: qwen3-coder:480b-cloud
  • Server Port: 5001

03 Core Capabilities

Vector Search: Local FAISS implementation for millisecond retrieval latency.

Local Embeddings: MiniLM-L6-v2 local model (zero cost).

Cloud LLM: High-performance reasoning via Ollama Cloud API.

Universal Ingestion: Native support for PDF and Plaintext.

04 Technical Architecture

Built with a modular RAG pipeline that separates document ingestion from the reasoning engine.

Flask LangChain FAISS Ollama Cloud HuggingFace Vanilla JS

05 Fixes

If port is busy, change port=5001 in app.py.

Run start_app.cmd for one-click launch.

06 Logic

Ingestion → Chunking → Embedding → FAISS Store → Retrieve → Prompt Enhance → LLM Generate.
